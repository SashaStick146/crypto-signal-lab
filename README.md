# Crypto Signal Lab 📡

Самообучающаяся «лаборатория сигналов» по крипто-рынку. Собирает данные с
биржи Bybit (бесплатно, без ключей), ищет сетапы (всплеск объёма, экстремум
фандинга, перекос лонг/шорт, RSI, пробои), а затем САМА проверяет, что было с
ценой через 24ч, и ведёт послужной список каждого типа сигнала. Чем дольше
работает — тем точнее: текущие сетапы ранжируются по доказанной исторической силе.

> Это аналитический инструмент, а НЕ инвестиционный совет.

## Что внутри
```
run_update.py            ← один цикл: сбор → сигналы → запись → оценка → отчёт → алерт
src/bybit.py             ← данные с Bybit (свечи, фандинг, лонг/шорт)
src/collect.py           ← список монет + сбор   ← СПИСОК МОНЕТ ЗДЕСЬ (COINS)
src/indicators.py        ← RSI, z-объёма, импульс
src/signals.py           ← правила сетапов        ← ПОРОГИ СИГНАЛОВ ЗДЕСЬ
src/track.py             ← журнал + самооценка    ← ГОРИЗОНТ ОЦЕНКИ (HORIZON_H)
src/report.py            ← report.html
src/alerts.py            ← Telegram (по желанию)
src/db.py                ← база (SQLite)
.github/workflows/lab.yml← бесплатный запуск каждый час на GitHub
```

## Запуск на компьютере
```
pip install -r requirements.txt
python run_update.py
```
Открой `report.html`. Запусти несколько раз с интервалом в часы/дни — начнёт
копиться статистика, и раздел «Послужной список сигналов» оживёт.

## Бесплатно 24/7 на GitHub
1. Новый репозиторий (Public — Actions бесплатны без лимита).
2. Залей все файлы (включая папки `src` и `.github`).
3. Actions → включить workflows → запустить «Crypto Signal Lab».
4. Включи GitHub Pages (Settings → Pages → main / root) и открывай
   `https://<логин>.github.io/<репозиторий>/report.html`.

### Telegram-алерты (по желанию)
Создай бота у @BotFather, узнай chat id у @userinfobot, добавь секреты
`TELEGRAM_TOKEN` и `TELEGRAM_CHAT_ID` в Settings → Secrets and variables → Actions.
Алерт приходит про сетапы, у которых тип сигнала уже доказал силу
(n≥20 проверок и средняя доходность ≥1%).

## Как это «умнеет»
Каждый сигнал записывается с ценой входа. Через 24ч программа смотрит, куда
ушла цена, и считает исход в сторону прогноза. По мере накопления исходов у
каждого типа сигнала появляется честный винрейт и средняя доходность — и
слабые сигналы естественно опускаются, сильные поднимаются.

покраснеет с ошибкой сети — напиши, переключим источник данных (код к этому готов).
